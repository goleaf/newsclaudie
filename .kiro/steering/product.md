# Product Overview

## Laravel Blog News

A modern, accessible, and feature-rich Laravel blog application with a powerful admin interface. Enhanced fork of Laravel BlogKit with significant improvements to admin UX, accessibility, and developer experience.

## Core Purpose

Provides a complete blogging solution with:
- Public-facing blog with posts, categories, comments, and tags
- Comprehensive admin panel for content management
- Multi-language support (English, Spanish)
- Accessibility-first design (WCAG 2.1 compliant)
- SEO-friendly with semantic HTML and metadata

## Key Features

### Public Features
- Blog posts with Markdown support and syntax highlighting
- Category and tag-based organization
- User comments with email verification
- Responsive design with dark mode
- Localization support

### Admin Features
- Livewire Volt + Flux UI powered admin panel
- Real-time updates with optimistic UI
- Advanced search, filtering, and sorting
- Bulk actions with configurable limits
- Inline editing for quick updates
- Keyboard shortcuts for power users
- Full CRUD for posts, categories, comments, and users

### Developer Features
- Property-based testing for critical features
- TypeScript support for frontend code
- Comprehensive documentation in `docs/`
- Code quality tools (Pint, PHPStan, Rector)

## Target Users

- Content creators and bloggers
- Development teams building content platforms
- Developers learning modern Laravel patterns
